My expertise and certifications

I had the opportunity to work on GCP projects. Although this is the cloud provider I used the less, I passed the Cloud Security Engineer certification to improve my knowledge in securing this cloud provider (click on the text to check it).

A lot of mistakes can be made upon setting up IAM as it is more complex than on other cloud providers in my opinion. This is the cloud provider I use on my personal projects.
